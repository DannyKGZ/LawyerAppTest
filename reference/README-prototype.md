# LawyerApp

История правок юридических документов с ответственностью. Концепция, модель данных и спека — в команд­ной вики: `team-wiki/LawyerApp/` (раздел [[index]]).

## Статус

- **2026-06-10** — кликабельный **прототип** различающих экранов: `prototype.html`.

## Прототип — как открыть

`prototype.html` — самодостаточный файл (React + Tailwind через CDN). **Просто открой его в браузере** (двойной клик или `xdg-open prototype.html`). Установка/сборка не нужны; нужен только интернет (для загрузки React/Tailwind с CDN).

Что внутри (на мок-данных):
- **Реестр** договоров с поиском/фильтром и индикатором просрочки.
- **Страница проекта** (реквизиты + пакет документов: договор + приложение + ДС).
- **Таймлайн** документа по дате с маркерами версий (`RMK12.OSN.v3`) и **сайд-бай-сайд диффом** на уровне пункта.
- **Карточка-ЗРС** с «живыми» кнопками решения (Принято / Доработка / Отклонено) — статус меняется на месте.
- **«Как видит Claude»** — демо agent-native слоя: `get_state` («на чём остановились»), `resolve_document` (опознать файл по коду), `prepare_next_version`, `export.md`.

Это **прототип для «пощупать»**, не продакшн. Боевая версия — Next.js по спеке (`team-wiki/LawyerApp/wiki/frontend-conventions.md`, `api-contract.md`). Бэкенд — зона Эдема.
